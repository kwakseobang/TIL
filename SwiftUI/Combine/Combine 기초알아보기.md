 # Combine이란?

- 반응형 프로그래밍의 개념을 기반으로 개발된 프레임워크
- Apple(WWDC): 시간 경과에 따른 값을 처리하기 위한 통합된 선언적 API
- Apple 공식 문서: 이벤트 처리 연산자들을 조합(combine)하여 비동기 작업 처리를 사용자화함

-apple 에서 통합됐다고 하는 이유는 iOS에서 비동기 인터페이스를 Publisher라는 동일한 인터페이스로 통합하였다.

## 반응형 프로그래밍?
- 데이터 스트림과 변화의 전파에 관련된 선언형 프로그래밍 패러다임

1. 데이터 스트림
   - 시간에 따라 만들어진 일련의 이벤트
  
2. 선언형 프로그래밍
   - "무엇"에 집중
   - 결과에 집중하도록 구체적인 행위를 감춤
   - 행위를 정의하고 그 행위에 데이터를 전달
   - 코드를 간결하게 하고 가독성 Up
  
  ## 반응형 프로그래밍 기반 패턴
  1. Observer Pattern
   - 주체(Subject): 옵저버 등록, 제거 및 알림 메커니즘 관리
   - 관찰자(Observer): 주체의 상태 변경을 감지하고 관련 작업 수행
