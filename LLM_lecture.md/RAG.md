# RAG
- 외부 데이터에서 적절한 문서 검색 및 답변을 생성하여 LLM 모델의 정확성과 신뢰성을 확장시키는 방법

# RAG 기본 개념 및 구성요소
- 검색 (Retrieval)
  - 학습된 데이터 X → 외부 데이터를 활용하여 사용자 쿼리와 가장 관련있는 데이터를 검색하는 과정
- 생성 (Generation)
  - 이전 검색 단계에서 도출한 가장 관련있는 정보와 사용자 쿼리를 함께 LLM 모델에 전달하여 최종
결과물을 생성

- How ? → 사용자의 질문(Query)과 DB에 존재하는 문서(Document)의 벡터 간 유사도 기반으로 관련 정보 추출
