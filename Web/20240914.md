# HTML 기본 

## HTML 
- HTML5 형식의파일은 항상 첫 줄에 문서형식 **<!DOCTYPE>** 을 선언하고 다음 줄부터 요소들의 모음으로 구성된다.
- <!DOCTYPE> 정보가 없으면 웹 브라우저에서 HTML 버전의 정보를 정확히 알 수 없다.
- HTML의 기본 구조
  - 문서형식 선언 다음에 항상 루트요소 html
  - head 와 body 로 구성, body 에 원하는 요소 구성.
  
- head 요소   
  - title 요소: 문서 제목
  - meta 요소
    - 문서 관리를 위한 메타 정보 표기
        - 문서 정보, 키워드, 저자 정보,언어, 인코딩 정보 등
    - 일반적으로 name 속성과 content 속성을 이용함

## 단락과 텍스트
- 제목: h1~h6의 요소
  - h1이 가장 큰 글자
  - 굵은 글자, 글 간력
- 단락: p 요소
  - 줄간격
- 줄 바꿈: br 요소
  - 단락 내에서 줄만 바꿀 때 사용
  - 단독 태그
- 가로줄: hr 요소
  - 문단 간 혹은 내용 사이에 구분을 확실히 하고자 할 때
- 작성된 형식 유지: pre 요소
  - 화면에 입력한 상태 그대로 보이고자 할 때
  - 공백문자: (빈칸, 탭, 줄바꿈)는 연속으로 여러 개 있어도 한 개로 취급한다.
- 단락 인용: blockquite 요소
  - 다른 글의 내용을 단락 단위로 인용하여 포함시킬 때
  - 들여쓰기로 구분하여 표시

## 다양한 텍스트 표현
- 텍스트 강조: em 요소
  - 기울여서 강조
- 강한 강조: strong 요소
  - 진하게 강조
- 작은 글자: small 요소
  - 약간 작은 글씨로 강조
- 하이라이트 효과: mark 요소
  - 형광펜 표시
- 위아래 첨자: sub 요소와 sup 요소

- 화면에서 출력 모양과 관련된 요소는 가급적 사용하지 않고 css 사용 권장.

## 목록 및 표 작성
- 순서 없는 목록: ul 요소
    - 목록 내에 나열된 항목을 나타내는 데에는 하위 요소인 li 요소 사용
- 순서 있는 목록: ol 요소
  - 각 항목 앞에는 순서를 나타내는 숫자 혹은 기호가 나타남
  -  하위 요소인 li 요소 사용
-  설명 목록: dl 요소
   -  사전과 같이 용어나 단어를 설명하는 설명 목록
   -  용어는 dt 요소에 적어주고, 그 설명은 dd 요소에 적는다
  
### 표의 기본 구성
- 표의 구성 요소: table, tr, th, td
  - tr 요소(table row): 하나의 행
  - td 요소(table data): 각 셀의 데이터
  - th 요소(table header): 제목, 중앙에 굵은 글씨

### 표의 구조적 표현
- 셀 합치기: rowspan, colspan
  - td 요소에 rowspan, colspan 속성 사용
- 표의 설명 제목: caption 요소
- 표의 머리줄, 몸체, 꼬리줄 표현: thead, tbody, tfoot 요소
  - thead: table 요소 내에서 머리줄에 해당하는 행의 집합
  - tfoot: table 요소 내에서 꼬리줄에 해당하는 행의 집합
  - tbody: thead와 tfoot에 속하지 않는 나머지 몸체 부분 행의 집합

### 문서 구조화
- 구조화 요소: 머리말, 꼬리말, 탐색 메뉴, 본문, 본문의 섹션, 옆줄 등 문서의 의미적인 구조
  - header, footer, nav, article, section, aside 요소들
  -  의미적인 구분만 할 뿐 모양은 구분하지 않는다.
  - 머리말: header 요소
    - 웹 문서에서 머리말 영역, 제목, 소개 등
  - 탐색 메뉴: nav 요소
    - 다른 웹문서나 문서 내의 다른 부분으로 이동하는 링크
  - 독립된 본문: article 요소
    - 웹문서에서 주요 내용을 가진 독립된 본문을 나타낼 때 사용.
    - 여러 개의 article 요소 가능
    - 각 article 요소 내에는 별도의 제목, header, footer, 여러 개의 section 요소가 포함될 수 있다.
  - 문서내 섹션 그룹: section 요소
    - 문서 내에서 절 단위 구분, 의미가 비슷한 그룹
  - 부수 정보: aside 요소
    - 본문의 내용과 구별되는 별개의 정보를 표현
    - 예를 들어, 웹 사이트의 배너 및 광고, 최신 댓글 목록 등을 표현할 수 있음.
  - 꼬리말: footer 요소
    - 웹문서의 꼬리말에 해당하는 저자 정보, 저작권 정보, 이용조건, 관련 링크 등
